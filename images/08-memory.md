# Image Briefs: Chapter 8 — Memory

## Hero Image

**Title / Slug:** `08-memory-hero`

**Description:**
A split image. Left side: a photograph of an old filing cabinet, slightly worn, with drawers open and folders visible, well-organized but showing age and use. Right side: the same filing cabinet, but the interior is abstract—files are floating, some overlapping, some fading, the organization has become fluid and uncertain. The metaphor is not perfect (memory is not literal filing), but it captures the idea of something that should be orderly but becomes reconstructed and fluid upon retrieval.

**Alternative approach:**
A high-angle shot of a person's face, eyes closed, with translucent neurons and synapses visible as an overlay around the head. The neurons are not connected in a grid; they are organic, branching, some brighter than others. At the bottom, a time-lapse or calendar effect showing hours, days, years flowing past. The visual emphasizes that memory is neural, embodied, and changes over time.

**Student should notice:**
The hero image should suggest that memory is not simple storage and retrieval. It is dynamic, reconstructive, and distributed across the brain. The image should feel more mysterious and fluid than the typical "video camera" metaphor.

## Concept 1: Three-Stage Model

**Image 1a: The Three-Stage Pipeline**

**Description:**
A horizontal flow diagram showing three boxes arranged left to right: Sensory Memory → Working Memory → Long-Term Memory. Each box contains visual elements representing its characteristics.

- **Sensory Memory:** A bright, momentary flash. Lightning bolt or burst. Label: "Vast capacity. < 1 second duration."
- **Working Memory:** A small, illuminated workspace (like a desk with a lamp). Label: "~4 chunks. 15-30 seconds."
- **Long-Term Memory:** A large archive or library, organized but with some items faded or overlapping in the back. Label: "Unlimited (effectively). Decades or lifetime."

Below each box, show the fate of unattended information: an arrow going down to a "discarded" or "faded" state.

**Student should notice:**
The trade-off between capacity and duration is visually apparent. Sensory memory is bright but fleeting. Working memory is accessible but tiny. Long-term memory is vast but not always accessible. The discarded pathway emphasizes encoding failure as a major cause of forgetting.

---

**Image 1b: The Forgetting Curve (Ebbinghaus)**

**Description:**
A graph with time on the x-axis (20 minutes, 1 hour, 1 day, 7 days, 30 days) and retention percentage on the y-axis (0-100%). A single curve drops steeply from 100% at time 0, reaching about 50% at 20 minutes, about 30% at 1 day, and leveling off at about 20% by 30 days.

Add a second line showing what happens if you review the material at each major drop: the curve resets to near 100% and drops again, but more slowly the second time (less steep slope).

**Student should notice:**
Forgetting is rapid at first, then slows. Review (rehearsal) resets the curve and makes it less steep on the second cycle. This is why distributed practice (spacing your studying over time with reviews) is more effective than cramming.

## Concept 2: Explicit vs. Implicit Memory

**Image 2a: Dual Systems**

**Description:**
A Venn diagram is not ideal here. Better: two parallel columns, one labeled "Explicit Memory" and one labeled "Implicit Memory." Each branch into two subtypes.

**Explicit (top):**
- Episodic: A photograph or scene. Example: "My graduation, June 2020."
- Semantic: A fact card. Example: "Paris is the capital of France."

**Implicit (bottom):**
- Procedural: Hands on a bicycle. Example: "How to ride a bike."
- Priming & Conditioning: A network or associations. Example: "The smell of coffee reminds me of my parent's kitchen" (emotional association).

**Labels:**
- Explicit: "Conscious, reportable, hippocampus-dependent."
- Implicit: "Unconscious, demonstrated in behavior, cerebellum & basal ganglia."

**Student should notice:**
Memory is not one system. Explicit and implicit are dissociable. H.M. lost explicit but retained implicit. This explains why he could improve at puzzles without remembering practicing them.

**Description:**
Two side-by-side portrait panels (stylized, not photorealistic).

- **H.M. (left):** Asking "Who are you?" (dialogue bubble). Caption: "Cannot form new explicit memories. Cannot recognize you if you leave the room and return."
- **Clive Wearing (right):** At a piano, hands playing with obvious skill. Caption: "Cannot remember his life. But his hands remember music. Procedural memory intact."

Below each: A small icon showing which memory systems are intact (checkmark) and which are damaged (X). H.M. has X for explicit, checkmark for implicit. Same for Clive.

**Student should notice:**
Both patients reveal that memory is modular. Different systems can be damaged independently. This tells us something about the brain's architecture—different structures, different purposes.

## Concept 3: Forgetting and Distortion

**Image 3a: The Loftus Misinformation Effect**

**Description:**
Three panels showing the experiment:

1. **Panel 1:** A car accident image (simplistic; a stylized crash).
2. **Panel 2 (left branch):** "Smashed?" in dialogue. Shows higher speed estimates (~40 mph). Participants' memory of the impact is stronger.
3. **Panel 2 (right branch):** "Contacted?" in dialogue. Shows lower speed estimates (~30 mph). Participants' memory is weaker.
4. **Panel 3:** One week later, both groups are asked: "Did you see broken glass?" The "smashed" group shows ~32% saying yes (X); the "contacted" group shows ~14% saying yes. But there was no glass.

**Student should notice:**
A single word changed not just the estimate but the memory itself. This is not a failure of memory. This is how memory works: reconstructive, influenced by new information presented after the event.

---

**Image 3b: The Ronald Cotton Case Timeline**

**Description:**
A horizontal timeline with key moments:

- **1984, June:** Jennifer Thompson is raped. Tries to memorize attacker's face.
- **Police show photos:** Thompson says "I think it's the guy" (wavering). Detective says "You did great" (reinforcement).
- **Lineup:** Thompson says "looks most like him" (still wavering). Confidence increases over time.
- **Trial:** Thompson testifies with complete certainty. Ronald Cotton is convicted.
- **1995 (11 years later):** DNA exonerates Cotton.

**Key visual:** Show confidence increasing over the timeline (arrow going up) while accuracy remains uncertain (question mark). The gap between confidence and accuracy widens over time.

**Student should notice:**
Confidence does not equal accuracy. Cotton was innocent, but Thompson's certainty grew as she rehearsed her memory, was given feedback, and testified. Each cycle of remembering reinforced the false version. Vivid certainty is not evidence.

**Image 3c: Schacter's Seven Sins of Memory (Table)**

**Description:**
A visual representation of the table from the source material, redesigned for clarity.

**Three categories (three columns):**
- **Forgetting (transience, absentmindedness, blocking)**
- **Distortion (misattribution, suggestibility, bias)**
- **Intrusion (persistence)**

Each cell contains:
- The name of the sin.
- A brief, plain-language description.
- A simple icon or symbol.
- An everyday example.

**Example:**
- **Transience** = Memories fade with time. Icon: Fading photograph. Example: "Can't remember details of a book you read last year."

**Student should notice:**
Memory failures are not random. They follow patterns. Some (encoding failure, transience) are about information not staying. Some (misattribution, suggestibility, bias) are about information getting twisted. Some (persistence) are about information that won't go away. Understanding the category helps you understand the mechanism.

## Synthesis Section

**Image 4: The Reconstructive Memory**

**Description:**
A single image showing memory retrieval as reassembly, not playback.

**Left side:** A memory event (example: a birthday party). Multiple sensory elements scattered as separate pieces:
- A color (red balloon).
- A sound (clinking glasses).
- A smell (cake).
- A face (friend smiling).
- A time label (Saturday afternoon).
- An emotion (happy).

**Right side:** These elements are being reassembled in a brain (silhouette or icon). But the reassembly is not perfect. Some pieces are highlighted, others faded. Some pieces might be slightly distorted (a face is slightly different from what it was). An arrow labeled "retrieval" points from scattered elements to the reassembled memory.

**Caption:** "Every time you remember, you reassemble. Each reassembly is a chance to change the story."

**Student should notice:**
Memory is distributed. Retrieval is reconstruction. This is why it is vulnerable to distortion. It also explains why the same memory retrieved many times can gradually change—each reassembly can alter the pieces slightly.

## Overall Visual Strategy

- **Avoid video camera metaphor.** Do not show memory as a recording device with playback.
- **Emphasize distribution and network.** Show memory as scattered, connected, reassembled rather than stored intact.
- **Make trade-offs visible.** Use size, brightness, duration in graphics to show capacity vs. time, access vs. accuracy.
- **Use real cases.** H.M., Clive Wearing, Jennifer Thompson, Ronald Cotton are named individuals with real stories. Images can be stylized but should reference their actual narratives.
- **Show mechanisms, not just metaphors.** A filing cabinet is a metaphor (imperfect). A three-stage pipeline is more mechanistic. A reconstructive reassembly is the actual mechanism.
- **Keep complexity appropriate.** Undergraduate students should understand these images; they should not oversimplify the concepts.
